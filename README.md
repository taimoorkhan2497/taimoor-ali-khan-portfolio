# Taimoor Ali Khan Portfolio

A hiring-focused single-page portfolio built with Next.js, TypeScript, Tailwind CSS, and Lucide icons.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

```bash
npm run lint
npm run build
```

## Deployment

The app is optimized for Vercel. Import this repository into Vercel and use the default Next.js build settings:

- Build command: `npm run build`
- Output: handled by Next.js
- Install command: `npm install`

Netlify can also host it with the official Next.js runtime using the same build command.
